VOSON Dashboard

VOSON Dashboard is an R/Shiny application for collecting and analysing networks and associated text data. It builds on a number of R packages, in particular igraph (for network analysis) and VOSON SML.

VOSON SML is an R package that provides a suite of tools for collecting and constructing networks from social media data. It provides easy-to-use functions for collecting data across popular platforms (Twitter, YouTube and Reddit) and generating different types of networks for analysis.
